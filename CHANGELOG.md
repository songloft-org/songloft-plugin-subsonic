# Changelog

## [2.2.2] - 2026-07-08
### Fixed
- 修复通过「导入」入库的歌曲在全平台不显示歌词的问题：入库时未携带歌词 URL 字段，导致宿主 `/songs/{id}/lyric` 返回 404（songloft-org/songloft#254）。已导入的歌曲重新导入一次即可修复。

## [1.0.2] - 2026-06-02
### Fixed
- 修复了在网络模式下（非本地导入）播放 Subsonic 音乐时，无法显示歌词的问题（注入了 `getLyrics` API 代理节点）。

## [1.0.1] - 2026-05-30
### Added
- 初始化插件并支持基本多节点挂载与流媒体播放。
